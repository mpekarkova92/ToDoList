# 📝 Modern ToDoList (PHP & MySQL)

Jednoduchá a efektivní webová aplikace pro správu každodenních úkolů. Projekt byl vytvořen s důrazem na čistý kód, bezpečnost dat a moderní responzivní vzhled.

## 🚀 Hlavní funkce
* **Full CRUD**: Vytváření, čtení, úprava a mazání úkolů.
* **Status Management**: Možnost označit úkol jako hotový (vizuální odlišení pomocí přeškrtnutí).
* **Bezpečnost**: Ošetření vstupů pomocí PDO Prepared Statements (ochrana proti SQL injection) a XSS ochrana.
* **User Experience (HCI)**: Automatické zaměření (autofocus) na vstupní pole a optimalizace pro mobilní zařízení.

## 🛠️ Technologie
* **Backend**: PHP 8.x
* **Databáze**: MySQL (propojení přes PDO)
* **Frontend**: HTML5, CSS3 (Flexbox), FontAwesome ikony

## 📂 Databáze
Struktura databáze je k dispozici ve složce `/sql` v souboru `database.sql`. Pro zprovoznění stačí naimportovat do vašeho MySQL serveru.
